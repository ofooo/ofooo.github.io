gitlab使用

设置保护分支(master分支不能推送)

默认的master分支是收保护的，不能直接push

进入setting—>Repository—>Protected Branches 设置保护分支

设置issue模板

在项目repository的代码文件夹新建一个文件夹:

.gitlab/issue_templates 文件夹

.gitlab/issue_templates/.gitkeep (空文本)

.gitlab/issue_templates/bug.md

.gitlab/issue_templates/feature.md

labels配置

kind（类型）

  • kind/bug
  • kind/新需求

priority（优先级）

  • priority/紧急
  • priority/不紧急

size（工作量）：表示 issue 需要大约花费多少时间/精力，可以用来做简单的工作量评估参考。

  • size/0小
  • size/1中
  • size/2大

CRLF/LF/CR

三种换行模式：

模式 操作系统 缩写
CRLF windows \n\r
CR mac OSX \r
LR Linux \n

pycharm可以在状态栏显示换行符使用的模式，如下图：

点击 LF 可以切换文件的换行模式
