删除文件的git控制

  • 如果已经加到版本控制中（push或add过）：用该命令去除控制（不删除本地文件）

    • git rm -r –cached 文件路径

  • 如果刚删除caehed，或未add或push过：直接设置 .gitignore 忽略即可。

  • .git/info/exclude 该文件和.ignore格式相同，但是不会被提交，不会影响他人的忽略名单。
