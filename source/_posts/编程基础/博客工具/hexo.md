## 安装hexo

```bash
sudo npm install hexo-cli -g
```









## 部署hexo

```bash
# 进入blog目录
hexo init
# 启动服务器
hexo server
hexo server -p 8080  # -p 端口

# 生成静态文件
hexo generate   | hexo g
`
```
