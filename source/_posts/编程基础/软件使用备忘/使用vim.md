### 命令模式

|             |                      |      |
| ----------- | -------------------- | ---- |
| u           | 撤销操作                 |      |
| Ctrl+r      | 恢复上一步被撤销的操作          |      |
| yy          | 复制单行                 |      |
| p           | 粘贴（到当前行下面）           |      |
| <u>数字</u>yy | 复制当前行到下面N行（适用于少量行复制） |      |
| 6,9 co 12   | 把6至9行copy到12行下面      |      |
|             |                      |      |
