## npm最常用命令

```bash
# 安装模块, 并将模块名加入到package.json
npm install xxxxxx --save
```



## npm常用命令

```bash
# 查看已安装模块的版本(-g查看全局,否则查看本地)
npm ls jquery -g

npm view jquery version  #查看jquery的最新的版本
npm view jquery versions  #查看npm服务器上所有的jquery版本信息
npm info jquery          #查看npm服务器上所有的jquery版本信息
```
