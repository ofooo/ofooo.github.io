#### 迭代器 Iterator  

```python
i = iter(nums)   
next(i)  
迭代器没有长度，它们不能被索引。
惰性，只能使用一次，只能循环遍历一次。
可以无限长，因为不调用next时什么也不做，不占多余的内存
```

#### 生成器 Generator

```python
生成器是迭代器的子类。
g = (n**2 for n in nums)
生成器所使用的场景是迭代前不生成这些数据，迭代后不再需要这些数据。
如果需要多次使用的数据，不应使用生成器。
```
