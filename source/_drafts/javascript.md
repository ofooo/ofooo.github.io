### 网页中的JavaScript位置

1. ```<button onclick="脚本的文本内容"></button>```   按钮onclick属性
2. ```<a href="javascript:脚本的文本内容"></a>   ```  超链接href属性
3. ```<head><script>脚本内容</script></head> ```  内部脚本
4. ```<head><script src="脚本位置"></script></head> ```  引用脚本文件（若有src，内部脚本无效）
